### 🎓 For Students

<details>
<summary><b>📖 Student Workflow</b></summary>

1. **🏁 Getting Started**
   - Navigate to your project folder: `projects/[YOUR-INDEX]-[YOUR-RESEARCH-AREA]/`
   - Read guidelines in `docs/project_guidelines.md`

2. **📝 Research Phase**
   - Start with research proposal in `docs/research_proposal.md`
   - Use GitHub Issues for progress tracking

3. **💻 Development Phase**
   - Regular commits expected for continuous progress
   - Follow repository structure guidelines

4. **📊 Monitoring**
   - Weekly supervisor check-ins via issue comments
   - Track milestones using project boards

</details>
